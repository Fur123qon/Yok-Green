# Yok Green

Yok Green is a community-based website that promotes environmental awareness and supports social actions focused on reforestation and sustainability in Java.

## About the Project
Yok Green is designed to introduce a strong environmental community that actively participates in tree planting, forest restoration, and social environmental campaigns across Java.

The website showcases activities, news, and opportunities for people to join environmental actions.

## Features
- Community introduction
- Activity showcase
- News & environmental updates
- Join community section
- Responsive design for mobile

## Tech Stack
- HTML
- CSS
- JavaScript

## Future Development
- User registration system
- Activity management dashboard
- Donation system for environmental programs
- Community event calendar
- Integration with environmental data APIs
